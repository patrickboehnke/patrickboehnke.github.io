# What are Metrics?
**Metrics** are a concept in software telemetry that consists of a single numeric value tracked over time. They are identified by **Labels**, which are descriptive attributes (e.g., hostname, service name, etc).

```
{
  Example:
  CPU utilization on a web server
  Memory usage on a web server
  Page load times
}
```
